## 2. 데이터 정확도 체크

| 항목 | 기준 | 실패 시 |
|------|------|---------|
| 미장 가격 | changePct ≠ 0, sparkline 20개 이상 | P0 |
| 국장 가격 | changePct ≠ 0, price > 0 | P0 |
| 코인 가격 | change24h ≠ 0, price > 0 | P0 |
| 지수 (KOSPI/SPX/NDX 등) | value > 0, changePct 존재 | P1 |
| 환율 (KRW/USD) | 1200 ~ 1600 범위 | P1 |

```bash
# API 직접 검증
curl "https://market-dashboard-v2-mu.vercel.app/api/us-price?symbols=AAPL,NVDA,TSLA" | python3 -c "import json,sys; d=json.load(sys.stdin); [print(r['symbol'], r['price'], r['changePct'], len(r.get('sparkline',[]))) for r in d['results']]"
curl "https://market-dashboard-v2-mu.vercel.app/api/market-indices" | python3 -c "import json,sys; d=json.load(sys.stdin); [print(r['id'], r['value'], r['changePct']) for r in d['results']]"
```
