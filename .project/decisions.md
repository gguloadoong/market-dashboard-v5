## ADR-001: 탭 구조 B안 채택
- **날짜**: 2026-03-17
- **결정**: `[🔥 지금 핫한 것] [🇰🇷 국내] [🇺🇸 해외] [🪙 코인] [📊 ETF]`

## ADR-013: Vercel 수동 배포 게이트 (workflow_dispatch 전용)
- **날짜**: 2026-03-29
- **결정**: Vercel Git 통합 자동 배포 비활성화. 프로덕션 배포는 GitHub Actions `workflow_dispatch` 수동 트리거로만 실행.
- **근거**: Vercel Pro 6,000 builds/월을 여러 프로젝트(chimp-pick, signalplay, costock, aptner 등)가 공유. PR 머지마다 자동 배포 시 빌드 한도 소진 불가피. 작업 단위 완료 후 배포 1회가 원칙.
- **구현**:
  - `vercel.json` → `ignoreCommand: "exit 0"` (Vercel Git 통합 배포 항상 스킵)
  - `.github/workflows/vercel-deploy.yml` → `workflow_dispatch`만 허용 (push 트리거 없음)
- **⚠️ 복원 금지**: `ignoreCommand`를 `exit 1` 또는 조건부 로직으로 되돌리는 것은 이 ADR 위반. "비표준"이 아니라 의도된 설계다. Vercel 기본값(main 머지 = 자동 배포)은 개인 프로젝트 기준이며 다중 프로젝트 운영 환경에 맞지 않는다.
- **배포 방법**: GitHub → Actions 탭 → "Deploy to Vercel" → Run workflow 클릭

## ADR-014: 반복 회귀 방지 — 파일 삭제 원칙 + 아키텍처 테스트 도입
- **날짜**: 2026-03-26
- **컨텍스트**: Sprint 1~6 동안 동일 기능이 반복 회귀한 구조적 원인 분석
- **데이터**:
  - 관련뉴스/관련종목 알고리즘: 10+ 커밋으로 6회 이상 재작성
  - EarlySignalSection: 삭제→복원→재삭제 3사이클
  - EventCalendar(섹션): 4사이클 반복
  - 프로젝트 전체 자동화 테스트: 0개
- **근본 원인 3가지**:
  1. **좀비 파일** — 삭제된 기능의 .jsx 파일이 filesystem에 남아있으면 새 AI 세션이 재발견 후 재import
  2. **결정이 코드에 없음** — "이 컴포넌트 삭제" 결정이 회의록/커밋 메시지에만 존재, 강제력 없음
  3. **테스트 전무** — 행동 회귀가 누군가 직접 발견하기 전까지 invisible
- **결정**:
  1. **삭제 결정 즉시 파일 삭제** — 파일이 없으면 재추가 불가. "나중에 정리"는 없다
  2. **COMPONENT_CONTRACT.md 패턴** — 각 기능 디렉토리에 활성/삭제 컴포넌트 목록 유지
  3. **아키텍처 테스트** — 삭제 금지 컴포넌트가 import되면 CI 즉시 실패
  4. **로직 회귀 테스트** — 가장 자주 수정되는 함수(필터, 알고리즘)는 반드시 단위 테스트 보유
- **적용**: `src/components/home/HOME_CONTRACT.md` + `src/__tests__/home-layout.test.js` + `src/__tests__/news-filter.test.js` (#180 → PR #181)
- **다른 프로젝트 적용**: `.project/templates/` 참고

## ADR-016: Phase 8B 서비스 방향성 재정의 — "시그널 인텔리전스"
- **날짜**: 2026-04-06
- **컨텍스트**: 대표님 피드백 15건. Phase 8A가 디자인만 정리하고 서비스 본질(AI 시그널)을 놓침.
- **핵심 결정**:
  1. **서비스 정체성**: "3시장 통합 투자 시그널 인텔리전스" — 데이터가 아니라 시그널을 판다
  2. **우리만의 언어**: 전문용어 배제, 쉽고 위트있는 시그널 메시지 (TYPE_META.easyLabel 체계)
  3. **홈 재구성**: NotableMovers 복원, 관심종목 v2 위치로, 온도계+공포탐욕 통합
  4. **AI 토론 본질 개편**: 3라운드 채팅 → "살 이유 vs 조심할 이유" 2줄 요약
  5. **신규 시그널 6종**: Cross-Market, Sentiment Divergence, Smart Money 등 (기존 데이터로 구현)
  6. **텔레그램 고래알림**: Whale Alert 유료 전환 → 무료 대안(텔레그램 채널 구독) 탐색
  7. **섹터 이상치**: changePct cap ±50% 적용
- **원칙**: "디자인 리뉴얼 시 핵심 가치를 전달하는 컴포넌트를 보호한 후 나머지를 정리"
- **교훈**: ai-coding-pitfalls #12 (Direction Blindness), #13 (Jargon Trap) 등록
