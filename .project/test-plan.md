# 마켓레이더 테스트 계획

## 검증 원칙
- **실제 파일을 읽고 판단한다. 추측으로 이슈 제기 금지.**
- Critical 이슈 발견 시 배포 차단

## 심각도 분류

| 등급 | 기준 | 대응 |
|------|------|------|
| 🔴 Critical | 핵심 기능 불가, 런타임 크래시, 보안 취약점, API 키 노출 | 즉시 수정, 배포 차단 |
| 🟠 Major | 특정 조건 데이터 오류, 로딩/에러 상태 미처리, 필터 오작동 | 배포 전 수정 |
| 🟡 Minor | 접근성 위반, 모바일 레이아웃, 불필요한 리렌더링 | 다음 스프린트 |
| ✅ Pass | 정상 동작 확인 | - |

## 핵심 테스트 시나리오

### 데이터 신뢰성
- [ ] KOSPI 수치가 네이버 금융 대비 ±1% 이내
- [ ] 코인 가격이 업비트 앱 대비 ±0.5% 이내
- [ ] 데이터 타임스탬프가 실제 갱신 시간과 일치
- [ ] "지연 데이터" 배지가 적절히 표시

### 급등 감지
- [ ] +3% 이상 종목이 SurgeBanner에 10초 이내 표시
- [ ] +5% 이상 종목 카드에 🔥 배지 + 빨간 좌측선
- [ ] 전 시장 통합 TOP 무버 정렬 정확성

### 모바일 (375px)
- [ ] 홈 탭 1열 레이아웃 정상
- [ ] 뉴스 접근 가능 (하단 탭 또는 시트)
- [ ] 텍스트 overflow 없음
- [ ] 터치 타겟 최소 44px

### API fallback
- [ ] CoinGecko 실패 → Upbit fallback 동작
- [ ] Yahoo Finance 실패 → Stooq fallback 동작
- [ ] 뉴스 RSS 실패 → CryptoCompare fallback 동작
- [ ] 모든 API 실패 → 마지막 정상 데이터 유지 + 경고 표시

### 보안
- [ ] .env 파일이 .gitignore에 등록
- [ ] 코드에 API 키 하드코딩 없음
- [ ] XSS 취약점 없음 (dangerouslySetInnerHTML 미사용)
- [ ] CORS 설정 정상

### 성능
- [ ] 초기 로딩 2초 이내
- [ ] 종목 클릭 → 차트·뉴스 3초 이내
- [ ] 뉴스 로딩 2초 이내
- [ ] 불필요한 리렌더링 없음
