# 마켓레이더 브랜드 가이드

## 서비스 정체성

**"3시장 통합 투자 시그널 인텔리전스"**

마켓레이더는 국내주식, 미국주식, 암호화폐 3개 시장의 시그널을 실시간으로 탐지하여
바쁜 직장인 트레이더에게 "지금 뭐 봐야 하는지"를 알려주는 서비스입니다.

---

## 로고

### 심볼

레이더 파동 3개 호(arc)와 시그널 점(dot)으로 구성.
- 3개의 호 = 3개 시장 (국내, 미국, 코인)
- 시그널 점 = 포착된 인사이트
- 좌하단→우상단 방향 = 상승, 성장

### 파일

| 용도 | 파일 | 설명 |
|------|------|------|
| 메인 로고 | `public/logo.svg` | 심볼 + 워드마크 (헤더, 랜딩) |
| 파비콘 | `public/favicon.svg` | 블루 배경 + 흰색 심볼 |
| PWA 아이콘 | `public/icons/icon-192.png` | 192x192 래스터 |
| PWA 아이콘 | `public/icons/icon-512.png` | 512x512 래스터 |
| 마스크 아이콘 | `public/icons/icon-maskable.png` | 512x512, 안전 영역 확보 |

### 사용 규칙

- 심볼 최소 크기: 24x24px
- 로고 주변 여백: 심볼 너비의 50% 이상
- 배경이 어두울 때: 흰색 워드마크 사용
- 심볼 단독 사용 가능 (좁은 공간, 앱 아이콘 등)
- 심볼 회전, 색상 변경, 그라디언트 적용 금지

## 톤앤매너

### 성격

- **친근한**: 전문 용어를 쉬운 말로. "급등" 대신 "뭔가 심상치 않아요"
- **위트있는**: 딱딱한 금융 데이터에 가벼운 뉘앙스. 이모지 적극 활용
- **직관적**: 한눈에 파악. 복잡한 차트보다 "지금 이거 봐"

### DO

- 짧고 명확한 문장
- 구어체 허용 ("~해요" 체)
- 데이터는 숫자보다 의미 중심 ("3일 연속 상승중")
- 적절한 이모지 (과하지 않게)

### DON'T

- 전문 금융 용어 남발 (RSI, MACD를 설명 없이 쓰기)
- 경고/공포 유발 ("폭락!", "대폭 하락!")
- 투자 조언/추천으로 오해될 표현
- 과도한 느낌표, 이모지 도배

## 태그라인 제안

1. **"시그널이 보이면, 기회가 보인다"** — 핵심 가치 직접 전달
2. **"3시장, 한눈에. 시그널, 한발 앞서."** — 서비스 범위 + 차별점
3. **"놓치지 마, 지금 움직이는 것들"** — 친근한 구어체, 즉시성 강조
