## 우리가 존재하는 이유 (Why)

**"급상승 종목을 누구보다 빠르게 캐치"**

토스증권은 예쁘고 쉽지만 정보가 부족하다. 업비트/빗썸은 코인만 본다. 키움HTS는 정보는 많지만 복잡해서 5분 안에 시장을 파악하기 어렵다.

마켓레이더는 그 빈자리를 차지한다: **국장 + 미장 + 코인을 하나의 화면에서, Bloomberg Terminal의 밀도와 토스의 단순함 사이 어딘가에서.**

## 포지셔닝

```
           정보 밀도 (많음)
                 |
    키움HTS ─────┼───── 우리 목표 위치
                 |         (즉각적 판단 가능한 컨텍스트 밀도)
    ─────────────┼─────────────── 직관성 (쉬움)
                 |
    토스증권 ────┼───── 업비트/빗썸
                 |
           정보 밀도 (적음)
```

우리는 "데이터"가 아니라 **"시그널"**을 판다.

## TAM / SAM / SOM

- **TAM**: 한국 개인투자자 2,400만 명
- **SAM**: 실시간 데이터에 목마른 액티브 투자자 1% = **24만 명**
  - 매일 아침 시장 확인하는 직장인 트레이더
  - 국내+미장+코인 동시 보유자
  - 모바일에서 급등 알림 필요한 데이트레이더
- **SOM**: 초기 1년 내 목표 5,000 DAU (SAM의 2%)

## 경쟁 차별화 전략

| 경쟁사 | 우리와의 차이 |
|--------|--------------|
| 토스증권 | 국장만, 코인 없음, 미장 제한적, 급등 알림 약함 |
| 업비트/빗썸 | 코인만, 국장/미장 없음 |
| 키움HTS | 정보 많지만 복잡, 모바일 UX 열악, 코인 없음 |
| Yahoo Finance | 영어, 국장 약함, 국내 투자자 UX 미최적화 |

**우리의 무기:**
1. 3시장 통합 (국장+미장+코인) 한 화면
2. 실시간 급등 스포트라이트 — 이유(뉴스 컨텍스트)까지
3. 한국 사용자 UX (원화 표시, 전일 종가 기준, 빨강=상승)
