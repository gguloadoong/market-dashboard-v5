## 🟡 Phase 2: UX 재구조화 (P0/P1)

### ~~[P0-5] 홈 구조 개편 — "이유" 중심~~ ✅ 완료

- 새 구조:
  ```
  1. 핵심 시그널 (이유 포함, 1~3개)   ← NEW
  2. 내 관심종목 현황 (간략)
  3. 시장 지수
  4. 급등/급락 통합 탭 리스트
  5. 오늘의 핵심 뉴스 (5건)           ← NEW
  6. 고래/섹터 (접기)
  ```
