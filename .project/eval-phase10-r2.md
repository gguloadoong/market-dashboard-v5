## PASS 차단 잔여 이슈 (1.8점 부족)

### P0: 반드시 수정

1. **환율 하드코딩 불일치** — `whalePattern.js`의 `1466`과 `WhalePanel.jsx`의 `1450`. 금액 계산 결과가 1.1% 차이남. 금융 앱에서 같은 계산이 파일마다 다른 상수를 쓰는 것은 수용 불가. `constants/exchange.js`에 `KRW_USD_RATE` 단일 소스 정의 필요.

2. **매직 넘버 상수 추출** — `$10,000,000` 임계값, `30 * 60 * 1000` 윈도우, `60_000` 감지 주기. 최소한 파일 상단 `const`로 명명 필요. 이것만 해도 코드 품질 매직넘버 항목 3->4로 +2점(가중) 확보 가능.

### P1: 강하게 권장

3. **STABLECOIN_SYMBOLS 중복** — 3곳 정의를 `constants/` 단일 파일로 통합.
4. **aria-label 추가** — 공유 버튼, 강도 바. 접근성 항목 3->4로 +2점(가중) 확보 가능.

### PASS 달성 경로

위 1~2번만 수정하면:
- 코드 품질: 매직넘버 3->4, 타입안전성 3->4 = +2.7점(가중)
- 합계: 214.2 + 2.7 = 216.9 -> **90.4% PASS**
